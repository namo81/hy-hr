크로스브라우징 범위

  • - 크롬
  • - IE11 이상

css 리스트

  • - layout.css - 기본 레이아웃 및 요소 전체 스타일 (필수)
  • - contents.css - 각 페이지별 레이아웃 (필수) / 일반 팝업 페이지일 경우 제외 가능
  • - popup.css - 일반 팝업용 (일반 팝업 페이지일 경우 필수)
  • - calendar-ui.css - SKT 달력 플러그인용 CSS
  • - jquery.mCustomScrollbar.css - 스크롤 디자인용 플러그인 CSS

plug in 리스트

jquery.mousewheel.min.js - SKT HR 사이트 접속 환경이 외부 링크를 차단함. 하여 스크롤 디자인에 필요한 추가 js.

  • - jquery-1.12.4.min.js - 기본 Jquery (필수)
  • - gnbmenu-v1.0.js - gnb 및 lnb 기능(필수)
  • - banner-tx.js - 상단 공지사항 배너기능(필수)
  • - layerpop.js - 레이어 팝업 기능
  • - jquery.mousewheel.min.js - 스크롤 플러그인에 필요한 js
  • - jquery.mCustomScrollbar.js - 스크롤 디자인용 플러그인
  • - calendar.js - SKT 달력 플러그인
  • - common.js - 공통 js ( input 디자인 / tabmenu 기능 / toggle 기능 / 말풍선 기능 ) (필수)

JS파일 적용 순서
jquery-1.12.4.min.js > gnb, banner, 레이어팝업 등 > common.js
jquery 파일과 common.js 파일 사이에 플러그인 파일들이 오도록.

공통요소

TASK Step

TASK 진행상황을 위한 step 표기.

기본 't-step' 클래스에 'step1 ~ step6' 까지 클래스 적용

태그는 span 이 아니어도 무관함.

TAKS작성 승인요청 반려 진행중 신고완료 피드백완료
<span class="t-step step1">TAKS작성</span> <span class="t-step step2">승인요청</span> <span class="t-step step3">반려</span> <span class="t-step step4">진행중</span> <span class="t-step step5">신고완료</span> <span class="t-step step6">피드백완료</span>

TASK 구분 아이콘

  • 개인 TASK / 협업 TASK 구분 아이콘
  • kind-task 가 기본이며, 'with' 클래스 추가 시 협업 TASK 아이콘으로 변경됨.
  • 텍스트 없이 아이콘만 사용되거나, 아이콘 위치를 우측으로 변경시 CSS 설정 필요.
  • 아이콘 이미지 사이즈 : 가로 24px / 세로 15px
  • 태그는 span 이 아니어도 무관함.
개인 TASK 협업 TASK
<span class="kind-task">개인 TASK</span> <span class="kind-task with">협업 TASK</span>

사용자 이미지

  • 사이트 내 사용자 이미지 사용시.
  • 이미지 사이즈 : 가로 28px / 세로 28px
  • 퍼블리싱 단계에서 원형으로 컷팅 하므로, 이미지는 원형이 아니어도 무관함.
  • 태그는 span 이 아니어도 무관함.

말풍선

버튼(btn-balloon) + 말풍선 컨텐츠(cnt-balloon) 구조

컨텐츠는 버튼 하단에 맞추어 자동으로 조정

버튼의 속성값 중 data-top / data-left 값을 추가하면 컨텐츠 위치 조정됨.

말풍선 위쪽 화살표는 after 요소에 border 로 생성. 디자인 변경시 CSS 수정 및 제거.

말풍선 내용 입니다.
